# departmentStaff

The program serves the data regarding company employees.

- The programm funcionality:

- Add employee
- Edit employee
- delete employee
- Show the list of employees

Tools used:
Java 1.7
Apache Tomcat 7.0
Servlets/JSP
MySQL 5.1
Apache Maven 3.3
JDBC
Hibernate (as the second way to connect to the DB)
